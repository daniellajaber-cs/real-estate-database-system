# Real Estate Database Management System

This project is a database design developed as part of a university Database Systems course.

## Description
The system models a real estate platform that manages properties, agents, buyers, listings, and transactions.
It includes ER/EER modeling, relational schema design, and SQL implementation.

## Components
- ER / EER Diagram
- Relational Schema
- SQL Table Creation Scripts
- SQL Queries

## Technologies Used
- MySQL
- SQL
- ER / EER Modeling

## Files
- ER_Diagram.png: ER/EER diagram of the system
- schema.sql: SQL script for creating tables
- queries.sql: SQL queries
- insert.sql: Sample data (optional)

This project is for academic purposes.
